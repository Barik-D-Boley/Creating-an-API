  • Specific Menu Item or add /api/menu/*item id* onto the end of the url
  • Query Search or add /api/menu/v1/query? with any combination of these parameters with an "&" in between each:
    • search=*word*
    • limit=*num*
    • minCal=*num*
    • maxCal=*num*
    • minPrice=*num*
    • maxPrice=*num*
